# The Essential Five

There are thousands of resources for every topic, and most people confuse
*collecting* resources with *learning*. They bookmark twelve courses, buy three
books, follow ten newsletters — and learn nothing, because they never actually
start. The fix is ruthless curation: a small set of the best resources, in the
right order, with a path to follow immediately.

Pick five. Five is enough to learn anything to a useful level and few enough to
actually finish. More than five and the learner is back to hoarding.

This feeds the **Sequence** pillar: the output is not just *which* five resources,
but the *order* to use them and a 7-day path — a defined sequence the learner can
start today instead of researching forever.

## Before you curate

- The specific topic and the learner's goal (this changes what "best" means — the
  best resource to *pass an exam* differs from the best to *build something*).
- Their level — beginner resources and advanced ones rarely overlap.
- Any constraints: budget (free only?), format preference (reading vs video), time.

If you're recommending current resources, prefer ones you're confident exist and
are well-regarded; if you're unsure something is still the best or still available,
say so rather than inventing a confident-sounding citation. Use web search if it's
available and the recommendations are time-sensitive.

## Output

Act as an expert learning curator. Find the **5 highest-leverage resources**. They
can be books, videos, courses, websites, newsletters, communities, or experts to
follow — whatever actually moves the needle for this topic.

For **each resource**, include:

1. **Resource name** (and where to find it).
2. **Type** — book, course, video, community, etc.
3. **Why it's worth their time** — the specific edge it has over alternatives.
4. **What part of the topic it covers** — so the five form a complete set, not five
   overlapping intros.
5. **Best learner type for it** — who it suits (and implicitly who it doesn't).
6. **Difficulty** — beginner, intermediate, or advanced.
7. **How to use it effectively** — skim? do every exercise? watch at 1.5x? This is
   often the difference between a resource working and not.
8. **One warning** — what to *not* waste time on within or around it.

Then:

- **Rank the five** in the best order to use them.
- **Give a 7-day learning path** that uses *only* these five — a concrete day-by-day
  so the learner starts today instead of researching forever.

## What makes curation good

- **The five don't overlap.** Each covers a distinct slice; together they're
  complete. Five beginner overviews is a failure.
- **It's opinionated.** "It depends, here are 20 options" is the noise you're
  removing. Commit to five and defend them.
- **The path is actionable now.** Day 1 should be doable today with no further
  shopping. The whole point is to stop collecting and start learning.

Focus on quality, clarity, and practical usefulness — the signal, not the noise.
